Relation

An immutable data structure representing a directional link between two entities.

Overview

The Relation class is a frozen dataclass used to define how different IDs within the system interact with one another. Since it is marked as frozen=True, instances are hashable and their attributes cannot be modified after creation.

Python Definition:
@dataclass(frozen=True) class Relation:

Attributes

Property Type Description
from_id string The unique identifier of the source entity.
to_id string The unique identifier of the target entity.
connection_type string The nature of the link (e.g., "reference", "depends_on", "contradicts").

Implementation Examples

Instantiating a Dependency

rel = Relation(from_id="A", to_id="B", connection_type="depends_on")

Creates a one-way dependency where A relies on B.

Logical Contradiction

rel = Relation(from_id="arg_1", to_id="arg_2", connection_type="contradicts")

Used to map conflicting data points in a knowledge graph.
