Class Entry

Entidad base para la gestión de contenidos y metadatos dinámicos.

Atributos

Campo Tipo Descripción
content str El cuerpo de texto principal de la entrada.
title Optional[str] Título opcional para identificar la entrada.
tags List[str] Lista de etiquetas para categorización (Default: empty list).
id str Identificador único universal (UUID v4).
created_at datetime Marca de tiempo de creación automática.
updated_at Optional[datetime] Fecha de la última modificación (Nulo si no se ha editado).
metadata Dict[str, Any] Diccionario flexible para datos adicionales.

Métodos

update_content(new_content: str)

Actualiza el contenido de la entrada y registra automáticamente la fecha de modificación en updated_at.

Nota: Los campos id y created_at se gestionan automáticamente mediante field(default_factory=...).
