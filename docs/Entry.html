Clase Entry Atributos Métodos Uso

Entry

Representación base de una entrada de datos con soporte para metadatos dinámicos.

Definición de Atributos

La clase utiliza dataclasses para gestionar el estado de la entrada.

Atributo Tipo Descripción
content str El cuerpo de texto principal de la entrada.
title Optional[str] Título opcional de la entrada (Default: None).
tags List[str] Lista de etiquetas para categorización.
id str Identificador único autogenerado (UUID4).
created_at datetime Marca de tiempo de creación automática.
metadata Dict[str, Any] Diccionario flexible para campos adicionales.

Métodos

update_content(new_content: str)

Actualiza el contenido principal de la instancia y refresca automáticamente la fecha en el campo updated_at.

Nota de implementación: El uso de field(default_factory=...) asegura que cada instancia tenga su propia copia única de listas y diccionarios.

Ejemplo de Uso

new_entry = Entry(
    content="Hola Mundo",
    title="Mi primera nota",
    metadata={"author": "Admin", "priority": "high"}
)

new_entry.update_content("Contenido actualizado")
