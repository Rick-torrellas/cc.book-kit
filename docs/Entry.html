Clase Entry

Entidad fundamental para la gestión de entradas de datos con soporte de metadatos.

Definición

La clase Entry utiliza @dataclass para representar una entrada de contenido enriquecida con identificadores únicos, marcas de tiempo y etiquetas.

Atributos

Atributo Tipo Descripción
content str El cuerpo principal del contenido de la entrada.
title Optional[str] Título opcional para la entrada.
tags List[str] Lista de etiquetas (tags) asociadas.
id str Identificador único generado mediante UUID4.
created_at datetime Fecha y hora de creación automática.
updated_at Optional[datetime] Fecha y hora de la última modificación.
metadata Dict[str, Any] Diccionario flexible para datos adicionales.

Métodos

update_content(new_content: str)

Actualiza el contenido principal de la entrada y registra automáticamente la fecha actual en el atributo updated_at.
